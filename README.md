# ひらがな・カタカナ れんしゅう

4〜8歳向けの、ひらがな・カタカナ書き順練習サイト。「まちがいOK・楽しむことが最優先」の探索型学習UX。

- **トップ → もじえらび → 五十音グリッド → かきじゅん（みる→まねする）→ クリア** の一方向フロー
- 全92字（ひらがな46＋カタカナ46）の**書き順アニメ**を内蔵
- タブレット横向き（基準 1280×800）を想定。依存ライブラリなしの単一 `index.html`

## 使い方
`index.html` をブラウザで開くだけ。ビルド不要。
