# Handoff: ひらがな・カタカナ 練習サイト（探索型・まちがいOK学習UX）

## Overview
4〜8歳の子供（＋保護者の補助）向けの、ひらがな・カタカナ練習アプリ。
設計思想は **「まちがいOK・楽しむことが最優先」の探索型学習**。正解／不正解の概念を持たず、
ネガティブな記号（❌✗）を一切使わず、触れるたびにポジティブなリアクションが起きる。

主な体験フロー：
**トップ → もじえらび（ひらがな/カタカナ）→ 五十音グリッド → かきじゅん（みる→まねする）→ クリア（ごほうび）**

対象デバイスは **タブレット横向き優先（基準 1280 × 800）**。タッチ前提で全ヒット領域が大きい。

---

## このローカル実装について
`../index.html` が本実装。Claude Design のプロトタイプ（React風 `support.js` ランタイム）を、
**依存ゼロのスタンドアロン HTML + バニラJS** へ移植したもの。ブラウザで `index.html` を開くだけで動作する。

- 1280×800 のステージを `transform:scale()` でウィンドウに等比フィット。
- 状態・筆画SVGアニメ・描画キャンバス・紙吹雪をすべてバニラJSで再現。
- フォント：Google Fonts（Zen Maru Gothic / Mochiy Pop One）。

## Screens / Views（基準 1280 × 800 px）

### 1. トップ（Home）
起動画面。マスコット円（「あ」＋顔パーツ）が出迎え、`▶ はじめる` で開始。
背景はラジアルグラデ、下端に緑の丘、雲☁️・星⭐が浮遊。

### 2. もじえらび（Select）
ひらがな（みどり）／ カタカナ（ピンク）の2枚の大カードを選ぶ。

### 3. 五十音グリッド（Grid）★採用案 = 「フォーカス」案C
左の大表示と右の表が**連動**。表マスをタップ → 左フォーカスの文字・絵文字・よみが即時切替。
表は**右から あ・か・さ…**（行を縦並び、列を右寄せ）。選択中マスは `pulse` で脈動。
ヘッダーのトグルで ひらがな⇄カタカナ（テーマ色＋表示かなが連動）。

### 4. かきじゅん（Write）= 「みる → まねする」2ステップ
- **みる**：本物の筆画ラインアニメ（あ行5文字、SVG中心線を1画ずつ描く＋番号バッジ＋🖌️追従）。
  あ行以外＋全カタカナは**フォールバック**（文字をワイプ表示）。
- **まねする**：`<canvas>` に指/マウスで自由描画。1ストロークごとに色が変わる。`けす` でクリア。
- フッター `かけたよ！ 🎉` → クリアへ。

### 5. クリア（Clear）= ごほうび演出
紙吹雪 ⭐🎉、⭐の中に達成文字＋顔、スタンプ累積（⭐数 = `done` の要素数）。
`↺ もういちど` / `つぎの もじへ ▶`。

## State Management（index.html の `state`）
- `screen`: `'home' | 'select' | 'grid' | 'write' | 'clear'`
- `type`: `'hira' | 'kata'`（テーマ色＋表示かなを決定）
- `sel`: 選択中文字のフラットindex（0–45）
- `done`: 習得済み集合（キー＝ひらがな文字）。⭐数 = `done` の要素数
- 筆画アニメは `svgTimer / fallbackTimer / strokeLens` 等の一時変数で管理（再描画と独立）。

## データモデル
全46文字を `[かな, 絵文字, よみ, 画数]` で保持（`H` 配列）。五十音表は10行×5列（や/わ行に空セル）。
カタカナは `conv()` でコードポイント +0x60 して導出。**よみ・絵文字・画数はひらがなと共有**。

### 筆画データ（全文字対応済み）
- `STROKES`（ひらがな46字）/ `STROKES_KATA`（カタカナ46字）= **`かな → { paths:[中心線SVGパス…], starts:[[x,y]…] }`**（viewBox 0–100）。フォント字形に重ねる軌跡＋番号矢印用。
- `KVG`（全ひらがな46字）/ `KVG_KATA`（全カタカナ46字）= **KanjiVG実データ**（viewBox 0–109, 出典: KanjiVG/Ulrich Apel, CC BY-SA 3.0）。字形・番号・軌跡が同一ソースで完全一致する高精度モード。**全92字で accurate**（みるパネルは KanjiVG 字形＋番号＋軌跡を描画）。`STROKES`/`STROKES_KATA` はフォールバック用に残置。
- みるパネルのアニメは **再生ボタン無しの自動連続再生**（`startFlow`：①→②→③を `pathLength=1` の dashoffset で順に書き込み、ループ）。

## Design Tokens

### Color
- 画面背景: home グラデ / select `#F3FAFF` / grid `#EEF6FF` / write `#FFFBEC` / clear グラデ
- 文字色: `#5A5246` / サブ `#7E7563` `#8A8273` / 淡 `#A9BFD0` `#B0A892`
- テーマ（hira＝みどり）: main `#7FC96B` / dark `#5E9E4C` / soft `#EAF6E4` / deep `#3F6B2E` / mainSoft `#C7E8BC`
- テーマ（kata＝ピンク）: main `#FF8FAA` / dark `#E87D97` / soft `#FFEAF0` / deep `#B24E68` / mainSoft `#FFCEDB`
- CTAピンク `#FF9FB6` / 影 `#E87D97`。黄ボタン `#FFC85A` / 影 `#E0A833`
- 筆画の画別カラー: `#FF7FA1` / `#5A9BD4` / `#7FC96B` / `#F5A623`。ペン循環に `#B07FE0` 追加

### Typography
- 本文・文字グリフ = **Klee One**（教科書体, 400/600）。練習画面の KanjiVG 手書き字形と形が一致するため、全画面の表示文字を Klee One に統一。タイトル = **Mochiy Pop One**。フォールバック = Zen Maru Gothic。
- スケール: Display 120 / 文字ヒーロー 110–150 / タイトル 56–58 / H2 40–42 / 本文 24–34 / ラベル 18–20。最小15px。

### Radius / Shadow
- マス/タイル 15–30 / カード 26–40 / ボタン・ピル 999 / 文字枠 20–32。
- “ぷっくり”＝下方向のソリッド影（例 `0 9px 0 #E87D97`）。CTAは加えて `0 18px 28px rgba(...)`。
- 文字枠内側: `inset 0 0 0 4px <mainSoft>`。最小タップ 44px。

## Assets
画像なし。各文字のイラストは **Unicode emoji**。本番では独自イラストSVGに差し替え可。
